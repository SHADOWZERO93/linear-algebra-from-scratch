# Linear Algebra from Scratch

Foundations of linear algebra implemented from scratch using NumPy.

## Topics Covered
- Vectors and scalars — geometric interpretation
- Vector addition and scalar multiplication
- Dot product — full derivation from Law of Cosines
- Unit vectors and normalization
- Angle between vectors
- Scalar and vector projections

## What I Learned
- Derived the dot product formula from geometry
- Proved the unit vector formula rigorously
- Implemented all operations in NumPy from scratch
